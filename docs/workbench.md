## 当前问题

1. 茶向量的 `0-10` 参考区间尚未由实测数据校准。
2. 中国茶标准、地方标准和地理标志信息还没有系统收录。
3. 图片素材还没有建立版权清单，当前首页使用远程公开图像链接作为占位。
4. 健康与茶疗内容需要严格证据分级，避免过度宣称。
5. 前沿科学部分需要补充近五年论文与研究机构。
6. GitHub Pages 需要在仓库设置中选择 `GitHub Actions` 作为发布源。
